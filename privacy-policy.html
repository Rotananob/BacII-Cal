គោលការណ៍ឯកជនភាព

ត្រឡប់ទៅការគណនា អំពីយើង គោលការណ៍ឯកជនភាព

🔒 គោលការណ៍ឯកជនភាព និងការគ្រប់គ្រងទិន្នន័យ

គោលការណ៍នេះពន្យល់ពីរបៀបដែល **ម៉ាស៊ីនគណនាលទ្ធផលបាក់ឌុប** (គេហទំព័រ ឬ Mobile App) គ្រប់គ្រងទិន្នន័យរបស់អ្នក។

⚡ សារសំខាន់៖ ម៉ាស៊ីនគណនានេះដំណើរការ **Offline**។ ទិន្នន័យដែលអ្នកបញ្ចូលមិនត្រូវបានបញ្ជូនទៅកាន់ Server ខាងក្រៅ ឬចែករំលែកជាមួយភាគីទីបីឡើយ។

១. ប្រភពទិន្នន័យ និងការប្រមូល

យើងប្រមូលតែទិន្នន័យដែលអ្នកបញ្ចូលដោយផ្ទាល់ទៅក្នុងប្រអប់ Input ប៉ុណ្ណោះ៖

  • ឈ្មោះអ្នកប្រើប្រាស់ (Username): ប្រើសម្រាប់បង្កើតសារជូនដំណឹងផ្ទាល់ខ្លួន (ឧទាហរណ៍៖ អបអរសាទរ រតនា !)។
  • **ពិន្ទុតាមមុខវិជ្ជា៖** ប្រើសម្រាប់តែការគណនាប៉ុណ្ណោះ។

២. ការប្រើប្រាស់ និងការរក្សាទុកទិន្នន័យ

ទិន្នន័យទាំងអស់របស់អ្នកត្រូវបានគ្រប់គ្រងដោយផ្ទាល់នៅលើឧបករណ៍របស់អ្នក (Client-Side)។

  • ដំណើរការ: ការគណនាទាំងអស់កើតឡើងនៅក្នុង Browser ឬ Mobile App របស់អ្នក។
  • ការរក្សាទុកបណ្តោះអាសន្ន: ឈ្មោះអ្នកប្រើប្រាស់ និងពិន្ទុរបស់អ្នកអាចត្រូវបានរក្សាទុកក្នុង **Local Storage** របស់ Browser/App ជាបណ្តោះអាសន្ន ដើម្បីឱ្យអ្នកមិនចាំបាច់បញ្ចូលឈ្មោះម្តងទៀតបន្ទាប់ពីចុច Clear Data។
  • អាយុកាលទិន្នន័យ: ទិន្នន័យត្រូវបានលុបចោលនៅពេលអ្នកបិទ Browser/App (Session End) ឬនៅពេលអ្នកលុបទិន្នន័យ/Cache របស់ App។

៣. ការអនុវត្តន៍លើ Mobile App (WebView)

នៅពេលប្រើប្រាស់គេហទំព័រនេះតាមរយៈ Mobile App (ដូចជា Android/iOS WebView) គោលការណ៍នេះនៅតែអនុវត្តដដែល។ ព័ត៌មានដែលអ្នកបញ្ចូល:

  • មិនឆ្លងកាត់ Server របស់ App ឡើយ។
  • មិនត្រូវបានតាមដាន ដោយ App Owner ឡើយ។

⚠️ កម្មវិធីរបស់យើងដាច់ខាត១០០%មិនប្រើប្រាស់ GPS, Camera, ឬ Contacts របស់អ្នកឡើយ។

៤. ការផ្លាស់ប្តូរគោលការណ៍

យើងរក្សាសិទ្ធិក្នុងការកែប្រែគោលការណ៍ឯកជនភាពនេះគ្រប់ពេលវេលា។ យើងនឹងជូនដំណឹងដល់អ្នកតាមរយៈការដាក់បង្ហាញការផ្លាស់ប្តូរនៅលើទំព័រនេះ។
